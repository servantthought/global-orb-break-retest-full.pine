# Global ORB Break & Retest - Full

Professional multi-session Opening Range Breakout (ORB) indicator for TradingView.

## Features
- Tracks **Asia, London, and New York** ORBs
- Break & Retest strategy with configurable confirmation (wick or close)
- Multiple historical ORBs with automatic cleanup
- Current ORB 50% and 100% extensions
- Daily & Weekly key levels (PDH/PDL, CWH/CWL, etc.)
- Approach alerts when price gets close to levels
- Clean visual design with status table

## How to Install
1. Copy the entire code from `Global ORB Break & Retest - Full.pine`
2. Open TradingView → Pine Editor
3. Paste the code and click "Add to Chart"
